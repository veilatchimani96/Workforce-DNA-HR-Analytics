# 👥 HR Analytics Dashboard – Workforce DNA: Decoding Employee Behaviour and Organisational Trends

An end-to-end HR Analytics project that analyzes employee demographics, attrition patterns, job satisfaction, and workforce trends using **Python, Excel, SQL, and Power BI**.

This project transforms raw HR data into meaningful business insights through data cleaning, exploratory data analysis (EDA), and interactive Power BI dashboards to support strategic HR decision-making. :contentReference[oaicite:0]{index=0}

---

# 📌 Project Overview

Employee attrition is one of the biggest challenges faced by organizations. This project analyzes workforce data to identify key factors influencing employee turnover and organizational performance.

The analysis includes:

- 👨‍💼 1,470 Employee Records
- 📉 Employee Attrition Analysis
- 📊 Workforce Demographics
- 💼 Department Performance
- 😊 Job Satisfaction Analysis
- 💰 Income Level Analysis
- 👥 Age Group & Gender Analysis

# 🧹 Data Preparation Pipeline

The dataset was cleaned and transformed before analysis.

## Data Cleaning

- Removed duplicate records.
- Handled missing values.
- Filled missing Monthly Income using Mean.
- Replaced missing categorical values with "Unknown".
- Standardized column names.

## Data Transformation

Created new calculated columns:

- Income Level
- Age Group
- Age Band

Prepared cleaned CSV dataset for Power BI Dashboard development. :contentReference[oaicite:2]{index=2}

# 💡 Key Insights

- Overall employee attrition rate is **16%**.
- Research & Development department recorded the highest attrition.
- Employees aged **26–35 years** represent the largest workforce group.
- Employees working overtime showed higher attrition.
- Monthly Income has a strong positive relationship with Total Working Years.
- Life Sciences employees experienced higher attrition compared to other education fields. :contentReference[oaicite:3]{index=3}
